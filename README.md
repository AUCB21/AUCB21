# 💫 About Me
Hi! I'm Augusto, a full-stack developer based in Buenos Aires 🇦🇷.

I'm currently working on **Tu Canchitapp** - a football field reservation, payment & client management platform built from the ground up with Next.js, PostgreSQL and TypeScript.

I'm always learning something new - right now diving deep into **Claude Code**, **Ollama**, and local AI tooling to build smarter, more autonomous development workflows. Not only that, also investigating on LLM integrations for ERP's.

I'm passionate about clean architecture, practical software, and building things that actually solve problems - from polished React interfaces to complex SAP HANA stored procedures.

🔭 Building with: Next.js · React · TypeScript · Django · PostgreSQL · SAP B1/HANA SQL  
🧠 Exploring: Claude Code · Ollama · Local AI tooling · LLM-powered automation · Cloud Hosting and Storage  
🛠️ ToolKit: VS Code · Antigravity · Claude Code · Hand-Made skills and workflows  
⚡ Always chasing: useful tools and zero unnecessary complexity.
